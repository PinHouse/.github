## Our Mission

누구나 자신에게 맞는 공공 임대주택을 쉽게 찾고,
더 나은 주거 환경에서 삶을 시작할 수 있도록 돕습니다.

## What We Do

**PinHouse**는 복잡한 공공 임대주택 정보를 한눈에 비교하고
개인 맞춤형 추천을 제공하는 혁신적인 주거 탐색 플랫폼입니다.

### Key Features

- **맞춤형 추천**: 사용자의 조건과 선호도 기반 스마트 매칭
- **직관적인 비교**: 복잡한 공공 임대 정보를 명확하게 시각화
- **빠른 탐색**: 조건별 필터링으로 원하는 집을 즉시 검색
- **투명한 정보**: 실시간 업데이트되는 정확한 임대주택 데이터

## Why PinHouse?

기존 공공 임대주택 정보는 흩어져 있고 이해하기 어려웠습니다.
우리는 이 문제를 해결하여 누구나 쉽게 접근할 수 있는 서비스를 만들고 있습니다.
